# DTSFO Blog Friends

这里是 DTSFO Blog 的友链申请仓库。

## 申请方式

1. 请先在自己的站点中加入 DTSFO Blog。
2. 在本仓库提交 Issue，或直接发 Pull Request。
3. 审核通过后，会同步到博客源码中的 `src/data/friends.json` 并上线展示。

## DTSFO Blog 信息

- 站名：DTSFO Blog
- 链接：https://blog.dtsfo.ccwu.cc/
- 头像：https://blog.dtsfo.ccwu.cc/assets/hero-portrait.png
- 简介：山海入墨，代码成篇。写技术，也写审美。
- 标签：技术 / AI / 运维 / 设计

## 推荐提交格式

```json
{
  "name": "站点名称",
  "url": "https://example.com/",
  "avatar": "https://example.com/avatar.png",
  "description": "一句话介绍你的站点",
  "tags": ["技术", "生活"]
}
```

## 收录偏好

- 独立域名或稳定托管，HTTPS 可正常访问。
- 至少有几篇原创内容，不只做导航或聚合。
- 站点无强制弹窗、恶意跳转或大面积采集内容。
- 链接长期可访问；失效后会暂时下线，后续可重新补档。
